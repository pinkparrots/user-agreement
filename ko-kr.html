PINKPARROTS 사용자 데이터 계약 (이하 "P.U.D.A.")

효능 날짜: 서비스 종료 시까지

Pink Parrots가 제공하는 모든 서비스 및 서버 ("ConstaX"와 다른 관련된 서버)에 접근또는 사용할시에 귀하 ("유저") 는 이 사용자 계약에 동의 하는것으로 간주됩니다.
만약 이 사용자 계약에 동의하지 않을시 Pink Parrots가 제공하는 모든 서비스 및 서버를 이용할 수 없습니다.

1. 정의
- "Pink Parrots", "당사" 는 Pink Parrots가 제공하고 있는 모든 서버의 관리자를 의미합니다.
- "서비스" 는 Pink Parrots가 제공하는 서버 ("ConstaX", 다른 관련된 서버 또는 기능) 을 의미합니다.
- "유저 데이터" 는 2번에서 설명하고 있는 유저를 식별하거나 식별할수 있는 정보를 의미합니다.

2. 당사가 수집하는 데이터
당사는 사용자에게서 다음과 같은 정보를 수집할수 있습니다:

- 사용자 IP
- 사용자 유저네임
- 디스코드 NameTag
- 디스코드 사용자 ID

3. 수집하는 데이터의 용도
귀하의 데이터가 다음과 같은 용도로 수집됩니다:

- 당사의 서비스의 사용성및 기능을 향상시키기 위해
- 남용, 사기 및 서비스 오용 방지를 위해
- 서비스간의 계정 연동및 인증을 위해 (예: 디스코드, 또는 다른 당사가 제공하는 서비스)
- 결제 정보및 계정 상태를 관리하기 위해

4. 제3자와 간의 데이터 공유
당사는 귀하의 개인 정보를 제3자와 판매, 대여, 공유를 어떠한 경우에도 하지 않습니다.
만약 이 정책이 변경이 된다면 귀하의 이메일 또는, 디스코드 개인 메세지로 변경되기 30일전에 알림이 전송될것입니다.

5. 사용자의 책임
귀하가 사용자로서 아래의 항목에 동의하는것으로 연루됩니다:

- 당사에게 가장 최신, 정확한 정보를 제공하는 것
- 당사가 제공하거나 다른 사용자에게 버그를 남용하여 피해를 주거나, 직접적으로 서버에 피해를 가는 행동을 하지 않는 것
- 당사의 서비스를 사용하는데 적용가능한 모든 법적을 따르는 것

6. 보안
귀하의 데이터는 업계 표준 암호화, 방화벽 보호 및 보안 프로토콜을 사용하여 안전한 서버에 저장됩니다.

7. 귀하의 권리
귀하는 다음과 같은 권리를 가질수 있습니다:

- 귀하의 개인정보에 접근또는 접근 요청하는 권리
- 귀하의 정보를 제거 요청하는 권리
- 언제든지 데이터 사용에 대한 동의를 철회할 수 있는 권리

 모든 요청을 하기 위해선 귀하가 공식 Discord 서버를 방문하여 문의 티켓을 열어 요청을 하거나, 아래에 서술된 이메일로 요청할수 있습니다.

8. 책임 제한
Pink Parrots는 다음에 대해 책임을 지지 않습니다.

- 당사가 통제할 수 없는 데이터 손실 또는 손상
- 서비스 중단 또는 다운타임
- 사용자 오용 또는 타사 서비스로 인한 손해

 9. 준거법
본 계약은 법 조항 충돌 여부와 관계없이 대한민국의 법률에 따라 규율됩니다.

10. 본 계약의 수정
당사는 본 계약을 업데이트할 권리를 보유합니다. 중요한 변경 사항은 시행 30일 전에 이메일 또는 Discord DM을 통해 고지됩니다. 서비스를 계속 사용하는 것은 업데이트된 약관에 동의하는 것으로 간주됩니다.
